1. Qu'est ce qu'un proxy ?

### Qu'est ce qu'un proxy ? * Un serveur **Mandaté** * Pour une application * Transmet les requêtes à internet * Souvent **HTML** * Fonctionne comme un "relai"

Un rôle compromis

Utillisation du protocol TCP/IP dans les réseaux locaux

-> Les routeurs assurent le rôle de relais


Les proxy sont utilisés pour d'autres fonctionalités

2. Les Fonctionnalités

Le Caching (proxy-cache)

  • Garde en mémoire les pages les plus visitées
  • Affichange rapide
  • Bande passante réduite


Pour que ceci fonctionne il est important que le proxy compare souvent ses données avec celles en ligne.

Le Logging / Tracking

  • Enregistre chaque connexion
  • Remplit des journaux d'activités (logs)
  • Enregistre Utilisateur + site accédé

Le filtrage

  • Analyse la Destination de la requête
  • Analyse le contenu des réponses serveur
  • Appilque un filtrage :
    • Liste blanche
    • Liste noire
    • Filtrage de contenu (mots-clés)

L'authentification

  • Intermédiaire indispensable
  • Authentification possible
  • Accès restreint aux ressources externes

3. Reverse Proxy

Qu'est ce qu'un reverse proxy ?

  • Proxy à l'envers
  • Permet aux utilisateurs d'internet d'acceder à certains serveurs du réseau interne
  • Bande passante réduite

Quel intérêt ?

  • Protège le réseau interne des attaques directes
  • Le cache soulage le serveur (accélérateur)
  • Réparti la charge
